# تطبيق بوت حجز شفتات ساموراي (Samurai Shift Bot)

هذا التطبيق هو مشروع أندرويد مفتوح المصدر يقوم بأتمتة عملية حجز نوبات التوصيل (الشفتات) من تطبيق "ساموراي" (delivery.samurai.android) باستخدام خدمة الوصول (Accessibility Service).

## كيف يعمل التطبيق؟
1. يستخدم التطبيق **Accessibility Service** لمراقبة الشاشة في تطبيق ساموراي.
2. يبحث التطبيق عن زر "حجز الشفت" (Book Shift) باستخدام المعرفات (IDs) التي تم استخراجها من التطبيق الأصلي (`btnBookShift`).
3. عند العثور على شفت متاح، يقوم التطبيق بالضغط عليه فوراً.
4. إذا لم يجد شفتات متاحة، يقوم التطبيق بعملية "سحب للأسفل" (Swipe Down) لتحديث القائمة بشكل دوري.

## متطلبات التشغيل
- هاتف أندرويد يعمل بنظام Android 5.0 (Lollipop) أو أحدث.
- تثبيت تطبيق "ساموراي" الأصلي.

## كيفية البناء (Build)
بما أن هذا الكود هو مشروع أندرويد خام، يجب عليك بناؤه باستخدام **Android Studio**:
1. قم بتحميل ملفات المشروع.
2. افتح Android Studio واختر "Open an existing project".
3. اختر مجلد `SamuraiBot`.
4. انتظر حتى ينتهي Gradle من تحميل المكتبات.
5. قم بتوصيل هاتفك أو استخدام محاكي، ثم اضغط على "Run".

## كيفية الاستخدام
1. بعد تثبيت التطبيق، افتحه واضغط على زر **"تفعيل خدمة الوصول"**.
2. ابحث عن "Samurai Bot" في قائمة الخدمات وقم بتفعيله.
3. افتح تطبيق ساموراي واذهب إلى صفحة "Working Shifts" أو "Book Shift".
4. سيبدأ البوت بالعمل تلقائياً والبحث عن الشفتات وحجزها.

## ملاحظة هامة
استخدام أدوات الأتمتة قد يخالف شروط استخدام تطبيق ساموراي. يرجى استخدام هذا التطبيق بمسؤولية وعلى مسؤوليتك الشخصية.
